Java | XML | UML

An Android game inspired by the Corornavirus.
As part of this project I also created UML & Screen Diagrams.

Video game success prediction

Python | Selenium | Pandas | API

A Data science (ML) project that attempts to predict a video game's likelihood of success before it launches.

  • The initial data was obtained using a web API.
  • Using Selenium, I wrote a web crawler to collect more data from various websites.
  • Cleansed data by removing irrelevant vectors, doing math calculations, and using scaling methods.
  • Examined correlations and outliers, and drew conclusions from a visual analysis of data.
